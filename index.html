Radioaktivitat Bot Detector

Discord anti-bot & airdrop autocollector detection with code-drop tools

Identify suspicious airdrop behavior (latency, typing patterns, repetition) and post reports to your chosen channel. Share codes safely using a modal + Reveal Code button (ephemeral & logged), including custom minimum wager notes for .com and .us.

Key features

  • Heuristics: latency, typing, CPM, smart-delay, repetition, reaction, entropy, humanization, account age
  • Tip-pattern signals: repeat gift ratio, reciprocity, amount-weighted ratio, time-correlation
  • Slash commands: /detector setup, /detector status, /detector threshold, /detector report, /detector codedrop, /detector tipstats
  • Reveal Code is private (ephemeral) and fully logged
  • Health endpoints: /metrics · /heuristics

Heuristics monitored

Fast latency
Replies too quickly after drop start (e.g., < 2s).
weight ~ 0.30
No typing
No “typing” signal before sending.
weight ~ 0.20
Unhuman CPM
Very high characters-per-minute while “typing”.
weight ~ 0.20
Smart-delay pattern
Message hits predictable offsets (e.g., ¼ of window).
weight ~ 0.15
Repetition
Same text repeated across recent messages.
weight ~ 0.15
Reaction too fast
Emoji reaction almost instantly after drop.
weight ~ 0.10
Low entropy
Message looks templated / copy-pasted (low Shannon entropy).
weight ~ 0.10
Low humanization
Lacks punctuation/emoji or natural token variation.
weight ~ 0.10
Very new account
Account age under N days (e.g., 7d).
weight ~ 0.10
High participation ratio
Shows up in a large fraction of drops (e.g., ≥ 60% after ≥ 8 samples).
secondary rule
Final score is the sum of active weights (capped at 1.0). Users are flagged when score ≥ threshold (default 0.6, configurable).
